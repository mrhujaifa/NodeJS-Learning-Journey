# Day 01 - CommonJS and EmsJS Modules

# CommonJS module system কী ?

- require() দিয়ে file import করা
- Destructuring import
- Module aliasing (str: newName)
- utils folder থেকে function import
- index.js দিয়ে shortcut import
- Multiple module একসাথে load করা

# ES Module (ESM) কী এবং কেন ব্যবহার করি

- import দিয়ে file import করা
- Destructuring import
- as ব্যবহার করে variable/function এর নাম পরিবর্তন করা
- .mjs extension ব্যবহার
- utils folder এর index.mjs থেকে shortcut import
- একাধিক file থেকে module import করা
- Modern JavaScript syntax ব্যবহার করে clean module structure তৈরি করা
